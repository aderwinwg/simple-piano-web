# [Simple Piano](https://aderwinwg.github.io/simple-piano-web/)

An interactive web-based piano application designed for learning basic piano skills. Play notes, explore demo songs, and work through structured lessons to build your musical foundation.

## Features

### 🎹 Interactive Piano Keyboard
- Full piano keyboard with 2 octaves (C3-C5)
- Visual feedback for pressed keys
- Computer keyboard mapping for easy play
- Touch and mouse support

### 🎼 Demo Songs
- Pre-loaded songs including:
  - Twinkle Twinkle Little Star
  - Ode to Joy
  - C Major Scale
  - Happy Birthday
  - Fur Elise (theme)
- Visual playback with key highlighting
- Stop all functionality

### 📚 Learn Section
- Structured piano lessons covering:
  - Musical alphabet
  - White and black keys
  - Sharps and flats
  - C major scale
  - Basic chords
- Interactive quizzes to test knowledge

### 🎯 Practice Section
- **Ear Training**: Listen to notes and identify them
- **Note Flashcards**: See note names and find them on piano
- **Scale Builder**: Build major and minor scales from any root note

### 🎛️ Controls
- Volume control
- Sustain mode toggle
- Label visibility toggle
- Stop all button

## How to Use

1. **Open the Application**
   - Open `index.html` in any modern web browser
   - No installation or server required

2. **Play the Piano**
   - Click piano keys with mouse or touch
   - Use computer keyboard (keys shown on piano keys)
   - Adjust volume with the slider

3. **Play Demo Songs**
   - Click any song button in the "Demo songs" section
   - Watch the visual playback
   - Press "Stop all" to stop playback

4. **Learn Piano**
   - Navigate to the "Learn" tab
   - Work through lessons sequentially
   - Take quizzes to test understanding

5. **Practice Skills**
   - Go to the "Practice" tab
   - Choose from ear training, flashcards, or scale builder
   - Improve your musical skills

## Technical Details

- **Pure HTML/CSS/JavaScript** - No frameworks or dependencies
- **Web Audio API** - For realistic piano sound synthesis
- **Responsive Design** - Works on desktop and mobile devices
- **Progressive Enhancement** - Graceful degradation on older browsers

## Browser Support

Works in all modern browsers that support the Web Audio API:
- Chrome 14+
- Firefox 25+
- Safari 6+
- Edge 12+
